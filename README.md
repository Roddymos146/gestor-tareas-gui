# Gestor de Tareas Personales - GUI con Tkinter

## Descripción del proyecto
Este proyecto consiste en el desarrollo de una aplicación de interfaz gráfica de usuario (GUI) utilizando Python y la librería Tkinter.

La aplicación fue diseñada con un propósito real: ayudar al usuario a organizar sus actividades pendientes mediante un gestor básico de tareas personales. A través de una interfaz sencilla e intuitiva, el usuario puede ingresar tareas, visualizarlas en una lista y eliminar elementos seleccionados.

## Objetivo
Desarrollar una aplicación GUI que permita interactuar con datos de manera visual, aplicando los conceptos básicos de interfaces gráficas de usuario.

## Contexto de uso real
En la vida cotidiana, muchas personas necesitan organizar sus actividades diarias para no olvidar pendientes importantes. Esta aplicación responde a esa necesidad permitiendo registrar tareas de forma rápida y visual.

## Funcionalidades
- Mostrar una ventana principal con título descriptivo.
- Ingresar una nueva tarea mediante un campo de texto.
- Agregar tareas a una lista visual con el botón **Agregar**.
- Eliminar una tarea seleccionada o limpiar el campo de entrada con el botón **Limpiar**.
- Responder a eventos generados por el usuario, como clics en botones.

## Componentes GUI utilizados
- `Label`
- `Entry`
- `Button`
- `Listbox`

## Librería utilizada
- Python
- Tkinter
